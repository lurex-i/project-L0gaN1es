Персональний помічник (Personal Assistant) розроблений командою League of Lizards.
Цей проект — консольна система для зберігання та взаємодії з записами адресної книги та нотатками. Програма дозволяє ефективно керувати контактами, планувати привітання з днем народження та вести структуровані записи з використанням тегів.

🚀 Основний функціонал
Робота з контактами (Адресна книга)
Зберігання даних: Додавання нових контактів із іменами, адресами, номерами телефонів, електронною поштою та датами народження.
Керування записником: Можливість редагувати та видаляти записи з книги контактів.
Пошук: Здійснення пошуку контактів за іменем та за номером телефону.
Нагадування про дні народження: Виведення списку контактів, у яких день народження настане через задану кількість днів від поточної дати.
Валідація: Перевірка правильності введеного номера телефону та email під час створення або редагування запису.

Робота з нотатками
Текстові нотатки: Створення, зберігання, редагування та видалення нотаток.
Система тегів: Додавання до нотаток ключових слів (тегів), що описують тему запису.
Пошук та сортування: Швидкий пошук за змістом нотаток або за ключовими словами (тегами).
Зберігання даних
Всі дані автоматично зберігаються на жорсткому диску у папці користувача.
Застосунок можна перезапускати без втрати внесеної інформації.

✨ Додатково реалізовані фічі
Ми розширили стандартні вимоги та додали наступні можливості для покращення користувацького досвіду:
Візуалізація інтерфейсу: Програма має текстове вітальне зображення (ASCII art) при запуску та спеціальну картинку при завершенні роботи.
Гнучкість управління: Реалізовано два режими роботи:
Командний режим: Введення прямих текстових команд (як передбачено основними вимогами).
Меню користувача: Навігація за допомогою цифр або букв для вибору потрібної дії.
Інтерактивна допомога: Додані підказки вводу, які допомагають користувачеві орієнтуватися, яку інформацію та в якому форматі потрібно ввести.
Покращена валідація: Крім базової перевірки формату email, додано валідацію домейну пошти, що забезпечує введення більш коректних даних.
